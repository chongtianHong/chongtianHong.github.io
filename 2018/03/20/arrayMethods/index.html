数组遍历方法forEach和map的区别

发表于 2018-03-20 | 分类于 开发
  • JavaScript中的forEach()和map()方法都是 ES5 为处理数组而新增的迭代方法，最主要的区别在于 map 方法返回一个新数组，而 forEach 方法没有返回值。
  • forEach()和map()都是遍历数组的方法，用法类似，但是还是有很大区别。

相同点：

  1. 都是按照顺序循环遍历数组中的每一项；
  2. 在遍历中执行匿名函数都可以接收三个参数，分别为：item（当前每一项）、index（索引值）、arr（原数组）；
  3. 执行的匿名函数中的this都指向Window对象；
  4. 只能遍历数组。

不同点：

  1. map()是根据遍历执行的匿名函数，对于原数组中的每个值产生一个对应的值，并返回一个新的数组，数组中的元素为原始数组调用函数处理后的值，存在一个映射关系，并且不会改变原数组，不会对空数组进行检测。

  2. forEach()只是按照顺序把数组中的元素传递给forEach()中的匿名函数使用，对于空数组则不会调用到匿名函数（对空数组进行检测）。
